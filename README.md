# cloud-notes-app_
A simple cloud-based notes app hosted on AWS S3 using static website hosting.
A simple and clean Notes Web App built using HTML, CSS, and JavaScript, deployed on Vercel and version-controlled via GitHub.

This project allows users to create, view, save, and delete notes directly in their browser using localStorage, ensuring notes remain even after refreshing the page.
